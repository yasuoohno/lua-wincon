InitialTextAttribute (new_attribute)
Get/Set initial text attribute.

Initial text attribute is used to restore text attribute when the dll is detaching from the process.
Its default value is taken when the DLL was attaching.

Parameters:

  • new_attribute optional number text attribute number, -1 or nil(do not change).
    The -1 value indicate that it will not restore text attribute when the dll is detaching.

Returns:

    number current initial text attribute number.
