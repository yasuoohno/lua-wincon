Module wincon

LuaWinCon is a Windows Console Utility for Lua.

This module allows some Windows console manipulation that change text and background colors and change codepage.

Info:

  • Author: Yasuo Ohno

InitialTextAttribute (attr)
Get/Set initial text attribute. Initial text attribute is used to restore text attribute on detached process. Its default value is taken when the dll was attached(module was loaded).

Parameters:

  • attr text attribute number or -1 or nil(do not change). The -1 value indicated that do not restore text attribute when the dll have been detached.

Returns:

    initial text attribute number.
